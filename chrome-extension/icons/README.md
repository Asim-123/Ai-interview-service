# Extension Icons

Place your extension icons here:

- `icon16.png` - 16x16 pixels
- `icon48.png` - 48x48 pixels  
- `icon128.png` - 128x128 pixels

You can create these using any image editor or online tool. Recommended design:
- Dark background with neon cyan/purple accent
- AI/robot/interview related icon
- Simple and recognizable

For now, you can use placeholder icons or generate them using online tools like:
- https://www.canva.com
- https://www.figma.com
- https://favicon.io

## Quick Placeholder

If you need quick placeholders, create simple colored squares:
- 16x16 with #00ffff background
- 48x48 with #00ffff background
- 128x128 with #00ffff background
